# Сценарий: Заказ пиццы через сайт

## Участники
- Клиент
- Система пиццерии

## Условия
- Клиент авторизован на сайте.
- В корзине есть хотя бы одна пицца.

## Шаги
1. Клиент переходит в корзину.
2. Нажимает кнопку **«Оформить заказ»**.
3. Система отображает форму с данными доставки.
4. Клиент заполняет поля:
   - Адрес
   - Способ оплаты
5. Нажимает **«Отправить заказ»**.
6. Система показывает сообщение: **«Заказ принят!»**.
7. Клиент получает email с подтверждением.

## Ожидаемый результат
Заказ успешно создан, данные сохранены в системе.
